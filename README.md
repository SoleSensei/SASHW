## Как делаем
Чтобы не было пересечений: заранее создаем файлик `readme.md` в папке с таском, где написано Кто-Что делает. 

__Команды__
```bash
pip install jupyter
git pull
jupyter /path_to_myfolders/Task_no
git push
```
